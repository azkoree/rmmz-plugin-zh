# rmmz-plugin-zh

一些RPG Maker MZ插件的自用汉化，大部分是一些小插件，因为大插件我怕不小心乱动导致整个插件都用不了了

~~似乎是很多人在做的工作，但是我做了也没什么问题吧~~

利用规约和原下载地址请参考帮助说明和原作者网站

本仓库汉化的插件基本都允许修改分发，特殊情况（付费/不允许分发）会进行标注，并只会提供参数和帮助文本的汉化，请自己替换
